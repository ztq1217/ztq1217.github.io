mondb启动命令以及参数

–port arg # 指定服务端口号，默认端口27017
–bind_ip arg # 绑定服务IP，若绑定127.0.0.1，则只能本机访问，不指定默认本地所有IP
–logpath arg # 指定MongoDB日志文件，注意是指定文件不是目录
–fork # 以守护进程的方式运行MongoDB，创建服务器进程
–dbpath arg # 指定数据库路径(常用)
–logappend # 使用追加的方式写日志

样例：linux指定ip启动mongodb

/usr/local/mongodb/bin/mongod –dbpath=/usr/local/mongodb/data –logpath=/usr/local/mongodb/logs –logappend –bind_ip=10.11.125.124 –port=27017 –fork

使用sh编译和启动后端程序

nohup java -jar /usr/local/qure.11.7.jar

其中nohup为记录日志的文件, java -jar为编译并启动.jar文件
