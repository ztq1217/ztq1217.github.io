maven个人向使用方法

发表于 2020-10-08 | 分类于 maven

可能只需要记住mvn package这一个吧？

配置相关

需要建一个变量名为 M2_HOME 的环境变量，变量值是maven的安装地址。
还要建一个变量名为 PATH 的环境变量，变量值是可以使用%M2_HOME%\bin

安装完成后可在cmd里输入mvn -v10:27 2020/10/12检查是否配置成功。

使用方法

在cmd中通过cd进入目标目录
执行mvn package

之后可以在target目录下找到生成的文件。

pom.xml设置
java文件：<packaging>jar</packaging>
web文件：<packaging>war</packaging>

目录说明

java项目目录名称 目录说明
src
src/main
src/main/java 存放java源文件
target 打包结果存放处
target/class 编译生成的class文件
target/maven-archiver 打包所需的配置文件
pom.xml maven工程的配置文件

建立与包名对应的目录文件。
比如包为package com.family168.maven2book.ch02;
则java源文件应放在
src/main/java/com/family168/maven2book/ch02/*.java

web项目目录名称 目录说明
src
src/main
src/main/webapp
src/main/WEB-INF 存放web源文件
target 打包结果存放处
pom.xml maven工程的配置文件
java 基础（二）
